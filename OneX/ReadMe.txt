Steps to import and run the project:
1. Import project as Maven Project into an IDE(preferablly Eclipse or InteliJ)
2. Ensure Chrome browser is version 126...., which is latest as of now
3. Download all dependancies for the project (Selenium and TestNG, see POM File)
4. Under src/test/java, run OneXAssesment.java as Java Application
5. The test framework should run smoothly and then generate or create screenshot

/testdata - directory with Test Data, we use Properties for this project
/webdriver - directoy containing chromedriver as our webdriver, can use geckodriver or any other
/screenshots - directory with screenshot captured after running the project
